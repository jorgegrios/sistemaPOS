# PCI DSS Checklist (high level)

This document summarizes steps to prepare for PCI compliance.

- Use tokenization and avoid storing PANs
- Use validated payment processors and certified SDKs
- Rotate API keys and store secrets in Vault
- Maintain network segmentation and logging
- Prepare evidence and run quarterly scans

This is a starting guide; follow official PCI DSS documentation and work with QSA for certification.
