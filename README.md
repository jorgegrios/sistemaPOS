# sistemaPOS — POS para Restaurantes (PWA)

Esqueleto inicial y documentación para un sistema POS avanzado orientado a restaurantes.

Contenido:
- backend/: Servidor Node.js + Express (TypeScript)
- openapi/: Spec OpenAPI 3.0
- docker-compose.yml: Entorno de desarrollo (Postgres, Redis, RabbitMQ)
- docs/: Documentación de integraciones y PCI

Objetivo: Proveer una base modular, escalable y lista para integraciones con procesadores de pago (Stripe, Square, Mercado Pago) y KDS / impresoras.
